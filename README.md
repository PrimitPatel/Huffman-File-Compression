# Huffman file compression

Huffman encoding to img and txt file

## Platform

```bash
Python 3.5 (or latest version)
```

## Required libraries

```
1. numpy
2. matplotlib
3. heapq
4. collections
5. bitarray 
```



## Output
Run Huffman_encoder.py and
type 'I/i' for image file compression or 'T/t' for text file compression.
